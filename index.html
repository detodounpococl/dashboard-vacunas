Dashboard listo para actualización por JSON

Este tablero prioriza indicadores útiles para decisión directiva: volumen total, concentración por grupo, comportamiento semanal, comparación interanual, alertas operativas y focos de gestión.

Puede cargar automáticamente un archivo influenza.json si está en la misma carpeta del HTML, o actualizar manualmente mediante el botón “Actualizar datos desde JSON”.

Formato esperado: objeto JSON con datasets y filas que contengan campos equivalentes a anio, origen, grupo, mes, semana, dosis. También acepta un arreglo plano de registros.
